# Octane

Usage-based pricing and monetization platform with a REST API for managing meters, pricing plans, customer entitlements, and real-time usage tracking for SaaS products. Octane enables infrastructure and SaaS companies to implement flexible pay-as-you-go billing similar to Snowflake and AWS.

## APIs

- **Octane REST API** — Manage meters, customers, pricing plans, subscriptions, and real-time usage measurements. Authentication via API key (Bearer token). SDKs available for Node.js, Python, and Go.
